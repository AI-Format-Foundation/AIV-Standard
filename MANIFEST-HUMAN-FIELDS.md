# OPTIONAL MANIFEST FIELDS FOR COPYRIGHT SUPPORT (AIV Standard)

These fields may be added to `manifest.json` for AIV:

"human_authorship_statement": "",
"human_signature": "",
"creator_email": "",
"editorial_notes": "",
"human_editing_steps": [],
"human_curated_output": true


These optional fields help creators establish a clear record of human involvement in AI-assisted video production.
